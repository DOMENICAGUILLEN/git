La programación es el proceso de crear un conjunto de instrucciones para que una computadora realice una tarea. Se realiza mediante el uso de algoritmos, que son secuencias de instrucciones ordenadas y finitas que deben seguirse para resolver un problema.

Tipos

Algunos de los principales paradigmas de programación son:

Programación declarativa.
Programación imperativa.
Programación estructurada.
Programación modular.
Programación orientada a objetos.
Programación orientada a eventos.

¿Cuál es la base de la programación?

Algoritmo

Son la secuencia de pasos lógicos que resuelven un problema, ¡son la base de la programación!
